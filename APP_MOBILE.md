# App iOS e Android

Este projeto agora está preparado como PWA.

## Instalar no Android

1. Abra o link publicado no Chrome.
2. Toque no menu de três pontos.
3. Toque em `Adicionar à tela inicial` ou `Instalar app`.

## Instalar no iPhone

1. Abra o link publicado no Safari.
2. Toque no botão de partilhar.
3. Toque em `Adicionar ao Ecrã Principal`.

## Publicar nas lojas

Para publicar na App Store e Google Play como aplicativo nativo, o próximo passo é empacotar este PWA com Capacitor.

Requisitos:

- Conta Apple Developer para iOS.
- Conta Google Play Console para Android.
- Projeto configurado com Capacitor.
- Ícones em PNG nos tamanhos exigidos pelas lojas.
- Backend real para email, cozinha/bar e pagamentos.
