# Acesso ao app da pizzaria

Link para outro computador ou celular na mesma rede Wi-Fi:

http://192.168.1.159:8000/index.html

Como iniciar pelo VS Code:

1. Abra esta pasta no VS Code.
2. Pressione `Ctrl+Shift+P`.
3. Procure por `Tasks: Run Task`.
4. Escolha `Iniciar app na rede`.
5. Deixe o terminal aberto enquanto outras pessoas usam o app.

Observacoes:

- O computador principal precisa ficar ligado.
- O outro computador precisa estar na mesma rede Wi-Fi.
- Se nao abrir, permita o Python no firewall do Windows.
- Se o IP do Wi-Fi mudar, rode `ipconfig` e use o novo IPv4 no link.
