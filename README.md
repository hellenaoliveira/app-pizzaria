# A Tal Pizza Brasileira

Aplicação responsiva para pedidos de mesa, com cadastro/login, seleção de mesa, cardápio, edição de pedido, envio para cozinha/bar, pedidos anteriores e finalização da conta.

## Publicar no GitHub Pages

1. Suba este projeto para um repositório no GitHub.
2. No GitHub, abra `Settings`.
3. Entre em `Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Escolha a branch `main` e a pasta `/root`.
6. Salve e aguarde o GitHub gerar o link.
