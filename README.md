# A Tal Pizza Brasileira

Aplicação web responsiva para gestão de pedidos em uma pizzaria. O sistema permite autenticação de usuários, seleção de mesa, visualização de cardápio e controle de pedidos até a finalização da conta.

---

## Descrição

Este projeto simula o fluxo de atendimento em um restaurante, com foco na organização de pedidos entre cliente, cozinha e caixa. Foi desenvolvido com o objetivo de consolidar conhecimentos em desenvolvimento front-end e estruturar uma aplicação com comportamento próximo a um sistema real.

---

## Funcionalidades

- Autenticação de usuários (login e cadastro)
- Seleção de mesa
- Visualização de cardápio (pizzas, porções e bebidas)
- Criação e edição de pedidos
- Envio de pedidos para preparo
- Consulta de pedidos anteriores
- Finalização da conta
